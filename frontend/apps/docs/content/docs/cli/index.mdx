  <Tab value="npm" copyable>npx @liam-hq/cli init</Tab>
  <Tab value="pnpm" copyable>pnpm dlx @liam-hq/cli init</Tab>

  <Tab value="bun" copyable>bun x @liam-hq/cli init</Tab>
</Tabs>

## Manual Setup

For more control over the ERD generation process, you can use the `erd` command directly:

### Basic Usage

Generate an ERD from your schema file using the following command:

<Tabs items={["npm", "pnpm", "yarn", "bun"]}>
  <Tab value="npm" copyable>{`npx @liam-hq/cli erd build --input <path|url> --format <format>`}</Tab>
  <Tab value="pnpm" copyable>{`pnpm dlx @liam-hq/cli erd build --input <path|url> --format <format>`}</Tab>
  <Tab value="yarn" copyable>{`yarn dlx @liam-hq/cli erd build --input <path|url> --format <format>`}</Tab>
  <Tab value="bun" copyable>{`bun x @liam-hq/cli erd build --input <path|url> --format <format>`}</Tab>
</Tabs>

This command processes your schema file and generates interactive ERD visualization files in the `dist` directory.

Once the ERD is generated, you can view it by serving the files using a local HTTP server:

<Tabs items={["npm", "pnpm", "yarn", "bun"]}>
  <Tab value="npm" copyable>npx http-server dist</Tab>
  <Tab value="pnpm" copyable>pnpm dlx http-server dist</Tab>
  <Tab value="yarn" copyable>yarn dlx http-server dist</Tab>
  <Tab value="bun" copyable>bun x http-server dist</Tab>
</Tabs>

The server will start and provide you with a local URL (typically http://localhost:8080) where you can view your ERD in a web browser.

You can use any hosting service of your choice to serve the generated files.

### Options

- `--input <path|url>`: Path to your schema file or URL
- `--format <format>`: Schema format specification

### From GitHub Public Repository

You can directly specify URLs to schema files stored in public GitHub repositories. Using raw URLs allows you to generate ERDs directly from remote schema files.

<Tabs items={["npm", "pnpm", "yarn", "bun"]}>

  <Tab value="pnpm" copyable>{`pnpm dlx @liam-hq/cli erd build --input https://github.com/user/repo/blob/main/examples/schema.sql --format postgresql`}</Tab>

### Output

The command generates a simple web application using [Vite](https://vite.dev/), which includes JavaScript, CSS, and HTML files, in the `dist` directory of your current working directory.

To view the generated ERD, serve the `dist` directory using any HTTP server:

<Tabs items={["npm", "pnpm", "yarn", "bun"]}>

  <Tab value="bun" copyable>bun x http-server dist</Tab>
