# インタラクティブなナビゲーション

Liam ERDは、最も複雑なデータベーススキーマでも簡単に探索できる直感的なナビゲーション機能を提供します。ダイアグラムインターフェースはスムーズなパンとズーム操作をサポートしており、全体の構造を把握しながら、特定の領域に焦点を当てることができます。

![パンとズーム操作のデモ](/images/content/docs/ui-features/browsing-your-schema/pan-and-zoom_2x.gif)

## パン（移動）操作

スキーマダイアグラムは以下の方法で移動できます：

- **マウス/トラックパッドのドラッグ**: キャンバス上の任意の場所をクリックしたままドラッグして表示を移動

## ズーム操作

表示したい詳細レベルに応じてズームイン/アウトができます：

- **マウスホイール**: 上にスクロールでズームイン、下にスクロールでズームアウト
- **トラックパッドのピンチ**: トラックパッド上でピンチイン/アウトでスムーズにズーム
- **タッチスクリーン**: タッチ対応デバイスで標準的なピンチジェスチャーを使用
- **キーボードショートカット**:
  - `Ctrl/Cmd +` でズームイン
  - `Ctrl/Cmd -` でズームアウト
  - `Ctrl/Cmd 0` でダイアグラム全体が表示されるようにズームをリセット

## フォーカス機能

大規模なスキーマを効率的にナビゲートするための機能：

- **ダブルクリック**: テーブルをダブルクリックすると、そのテーブルが中央に表示されるようにズーム
- **自動フォーカス**: リレーションシップのリンクをたどる際、関連するテーブルが自動的に表示されるように視点が調整

## パフォーマンス

ダイアグラムインターフェースは以下のような条件でも最適化されたパフォーマンスを発揮します：
- 100以上のテーブルでもスムーズに動作
- 複雑な関係性と接続
- 複数のユーザーが同時に操作

## 効率的なナビゲーションのヒント

1. **概要の活用**:
   - まずズームアウトして全体構造を把握
   - 徐々にズームインして特定の領域を詳しく調査

2. **キーボードとマウスの組み合わせ**:
   - `Space`キーを押しながらドラッグすると一時的にパンモードに
   - `Alt`キーを押しながらスクロールでズーム速度を調整

3. **タッチジェスチャー**:
   - 2本指スクロールでパン
   - ピンチジェスチャーでズーム
   - ダブルタップで特定の要素にフォーカス

これらのナビゲーション機能はReact Flowを利用して実装されており、大規模で複雑なデータベーススキーマでもスムーズで快適なユーザー体験を提供します。
